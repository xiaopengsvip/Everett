# 🌿 Everett Minds

> **Write · Think · Grow**  
> 用语言表达，用书写沉淀，用思考成长。

Everett Minds 不是一个普通的在线学习网站，而是一个**专注训练表达与思维的数字道场**。我们相信，人类的成长本质上是表达能力的提升，而表达来源于语言的精准、书写的专注以及思考的深度。

---

## 🧩 核心模块 (Core Modules)

Everett Minds 将个人成长拆解为三个核心维度的刻意练习：

### 🗣 Language · 语言表达
**“不只是学单词，而是学会表达。”**
*   **每日英文日记 (Daily Journal)**：用英语记录生活与感悟。
*   **跟读复述训练 (Shadowing)**：输入与输出的闭环，培养语感。
*   **观点辩论 (Debate)**：针对特定话题，构建正反方逻辑论据。
*   **真实场景模拟 (Scenario)**：在具体语境中激活词汇。

### 🖋 Ink · 书写艺术
**“写字，是一种专注的修行。”**
*   **字帖临摹 (Copybook)**：专注于汉字结构与笔画的练习。
*   **自由书写 (Free Writing)**：不拘泥于形式，感受书写的节奏与心流。
*   **古诗词抄写 (Poetry Copying)**：在抄写经典中体会文字的韵律美。
*   **字体排版设计 (Typography)**：探索文字的视觉呈现与美感。

### 🧠 Mind · 思维训练
**“你写下的，就是你思考的。”**
*   **心智模型画布 (Mental Model)**：套用经典模型分析复杂问题。
*   **第一性原理 (First Principles)**：剥离表象，直击事物本质。
*   **5 Whys 分析法 (5 Whys)**：连续追问，深挖问题的根本原因。
*   **决策矩阵 (Decision Matrix)**：理性的多维度决策分析工具。

---

## ✨ 核心特性 (Key Features)

*   🎨 **沉浸式艺术工作台**：精心设计的双侧边栏（Primary & Secondary Sidebar）布局，提供无干扰的专注学习环境。
*   📝 **全功能 Markdown 支持**：练习面板原生支持 GitHub Flavored Markdown (GFM)，让你的思考结构化呈现（支持表格、任务列表等）。
*   ⏱ **全局学习时长记录**：自动追踪并记录你的每一次心流时刻，在仪表盘（Dashboard）直观展示累计学习时长。
*   🌍 **双语无缝切换**：原生支持中文与英文界面，一键切换。
*   优雅的 UI/UX：基于 Tailwind CSS 与 Framer Motion 打造的流畅动画与高级排版（采用 Cormorant Garamond 衬线字体）。
